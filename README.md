# 📈 Customer Data Analysis

## 📌 Project Overview

Customer Data Analysis focuses on analyzing retail customer behavior across multiple shopping malls using SQL, Excel, and Power BI. The project identifies sales trends, customer segments, revenue performance, and business KPIs through interactive dashboards.

---

## 🎯 Business Problem

Retail organizations require data-driven insights to improve customer retention, increase sales, and monitor overall business performance.

---

## 🎯 Objectives

- Analyze customer purchase behavior
- Perform sales trend analysis
- Build KPI dashboards
- Segment customers
- Generate business recommendations

---

## 📊 Dataset

- Retail Customer Dataset
- 10,000+ Records
- Multiple Shopping Malls
- Sales Data (2021–2023)

---

## 🛠️ Tools & Technologies

- Power BI
